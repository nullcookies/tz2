# File upload using Laravel and Vue

### Installation

1. Clone repo

2. Change to directory

````
cd PROGECT_NAME
````   

3. Install dependencies

````
composer install
````

4. Copy .env file

```
cp .env.example .env
```

5. Modify `DB_*` value in `.env` with your database config.

6. Generate application key:

````
php artisan key:generate
````

7. Migrate
````
php artisan migrate
````
